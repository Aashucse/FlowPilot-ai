FlowPilot AI

An AI productivity assistant that helps users organize tasks clearly and practically.

Features:
- Chat-based task planning
- Simple and clean UI
- Fast responses using OpenRouter API

How to run:
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000

Tech stack:
- Python (Flask)
- HTML, CSS, JS
- OpenRouter API
